# book-pde
Учебник PDE.
Заметки, брошюрки и структурирование материала от команды людей, изучающих PDE (уравнения с частными производными).

Книга написана на jupyter-book (https://jupyterbook.org/)

# Get Started

Установить jupyter-book https://jupyterbook.org/stable/get-started/install/
В папке проекта запустить
```bash
jupyter book init
```

# Build
Как запустить сборку:
```bash
jupyter book build 
```
